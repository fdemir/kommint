# What is Kommint?

Kommint is the easiest way to add comments section to your application.

## Motivation

### Why we exists

- To help developers build better and user-centric products
- To help you enhance the user engagement
- To build a better community

### Why we build

The current alternatives are not customizable, hard to use, and hard to integrate to existing applications. Not adapted to the needs of the community.

### What's our vision

contribute to how user interactions and comments in applications should be.
